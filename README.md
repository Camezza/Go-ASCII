## Go ASCII
 This is a simple program written in `go` that turns images into ASCII art!
 
 ### Usage
 ```bash
 git clone https://github.com/Camezza/Go-ASCII.git
 cd Go-ASCII
 go run . <image path> <output directory path>
 ```
